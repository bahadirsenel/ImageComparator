# BILDJÄMFÖRARE - ANVÄNDARGUIDE

Välkommen till Bildjämförare! Denna applikation hjälper dig att hitta och hantera dubblettbilder i dina mappar.

═══════════════════════════════════════════════════════════

## 1. LÄGGA TILL MAPPAR
   • Klicka på knappen "Lägg till mapp" eller dra och släpp mappar i utdatalistan längst ner
   • Applikationen kommer att skanna efter bildfiler i den valda mappen
   • Du kan lägga till flera mappar för att jämföra bilder på olika platser

## 2. HITTA DUBLETTER
   • Klicka på "Hitta dubletter" för att starta jämförelseprocessen
   • Applikationen kommer att analysera bilder och hitta liknande eller dubblettfiler
   • Framsteg visas i förloppsindikatorn
   • Du kan pausa eller stoppa processen när som helst med respektive knappar

## 3. FÖRSTÅ RESULTAT
   • Dubblettpar visas i två listor sida vid sida
   • Färgkodade indikatorer visar konfidensnivåer:
     - Blå: Dublett
     - Röd: Hög konfidens för likhet
     - Gul: Medelhög konfidens för likhet
     - Grön: Låg konfidens för likhet
   • Klicka på valfri fil för att förhandsgranska den i mittenpanelen

## 4. SÖKFORMAT
   • Gå till Alternativ > Sökformat för att välja vilka bildtyper som ska skannas
   • Stödda format: JPEG, BMP, PNG, GIF, TIFF, ICO
   • Alla format är aktiverade som standard

## 5. RADERINGSALTERNATIV
   • Välj filer att radera med hjälp av kryssrutorna
   • "Radera valda": Tar bort markerade filer från ditt system
   • "Markera för radering": Markerar filer för senare radering (visas i rött)
   • "Ta bort från lista": Tar bort objekt från resultat utan att radera filer

## 6. RADERINGSMETOD
   • Gå till Alternativ > Raderingsmetod för att välja:
     - "Skicka till papperskorgen": Säkrare alternativ, tillåter filåterställning (standard)
     - "Radera permanent": Filer raderas permanent utan återställningsalternativ

## 7. FALSKA POSITIVA
   • Om två bilder är felaktigt matchade, välj dem och klicka på "Markera som falskt positiv"
   • Dessa par kommer att kommas ihåg och visas inte i framtida sökningar
   • För att rensa databasen för falska positiva, gå till Alternativ > Rensa databasen för falska positiva

## 8. SPRÅKALTERNATIV
   • Gå till Alternativ > Språk för att växla mellan 18 språk som stöds:
     English, Türkçe, 日本語, Español, Français, Deutsch, Italiano,
     Português (Brasil), Русский, 简体中文, 한국어, العربية, हिन्दी,
     Nederlands, Polski, Svenska, Norsk, Dansk
   • Gränssnittet uppdateras omedelbart

## 9. YTTERLIGARE ALTERNATIV
   • "Inkludera undermappar": Skannar alla undermappar i valda mappar
   • "Hoppa över filer med olika orientering": Ignorerar porträtt vs. landskapsmatchningar
   • "Hitta endast exakta dubletter": Hittar endast exakta dubblettfiler, liknande filer visas inte

## 10. SPARA OCH LADDA RESULTAT
    • Arkiv > Spara resultat: Sparar dina nuvarande fynd för att fortsätta senare
    • Arkiv > Ladda resultat: Laddar tidigare sparade resultat

## TIPS:
• Du kan zooma in/ut på förhandsgranskningsbilder med mushjulet eller +/- knapparna
• Högerklicka på filer för att öppna dem eller visa deras plats
• Använd tangentbordsnavigering (piltangenter) för att snabbt bläddra genom resultat
• Dubbelklicka på objekt för att öppna dem i din standardbildvisare

För mer information eller för att rapportera problem, besök projektets arkiv.
