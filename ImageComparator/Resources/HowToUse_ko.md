# 이미지 비교기 - 사용자 가이드

이미지 비교기에 오신 것을 환영합니다! 이 애플리케이션은 폴더에서 중복 이미지를 찾고 관리하는 데 도움을 줍니다.

═══════════════════════════════════════════════════════════

## 1. 폴더 추가
   • "폴더 추가" 버튼을 클릭하거나 하단의 출력 목록으로 폴더를 드래그 앤 드롭합니다
   • 애플리케이션은 선택한 폴더에서 이미지 파일을 검색합니다
   • 여러 폴더를 추가하여 다른 위치의 이미지를 비교할 수 있습니다

## 2. 중복 찾기
   • "중복 찾기"를 클릭하여 비교 프로세스를 시작합니다
   • 애플리케이션은 이미지를 분석하고 유사하거나 중복된 파일을 찾습니다
   • 진행 상황은 진행률 표시줄에 표시됩니다
   • 해당 버튼을 사용하여 언제든지 프로세스를 일시 중지하거나 중지할 수 있습니다

## 3. 결과 이해
   • 중복 쌍은 나란히 두 개의 목록으로 표시됩니다
   • 색상으로 구분된 표시기는 신뢰도 수준을 보여줍니다:
     - 파란색: 중복
     - 빨간색: 높은 유사성
     - 노란색: 중간 유사성
     - 녹색: 낮은 유사성
   • 중앙 패널에서 미리 보려면 파일을 클릭하세요

## 4. 검색 형식
   • 옵션 > 검색 형식으로 이동하여 스캔할 이미지 유형을 선택합니다
   • 지원되는 형식: JPEG, BMP, PNG, GIF, TIFF, ICO
   • 모든 형식이 기본적으로 활성화되어 있습니다

## 5. 삭제 옵션
   • 체크박스를 사용하여 삭제할 파일을 선택합니다
   • "선택 항목 삭제": 시스템에서 선택한 파일을 제거합니다
   • "삭제 표시": 나중에 삭제할 파일을 강조 표시합니다 (빨간색으로 표시됨)
   • "목록에서 제거": 파일을 삭제하지 않고 결과에서 항목을 제거합니다

## 6. 삭제 방법
   • 옵션 > 삭제 방법으로 이동하여 선택합니다:
     - "휴지통으로 보내기": 더 안전한 옵션, 파일 복구 허용 (기본값)
     - "영구 삭제": 복구 옵션 없이 파일이 영구적으로 삭제됩니다

## 7. 오탐지
   • 두 이미지가 잘못 일치하면 선택하고 "오탐지로 표시"를 클릭합니다
   • 이러한 쌍은 기억되며 향후 스캔에 나타나지 않습니다
   • 오탐지 데이터베이스를 지우려면 옵션 > 오탐지 데이터베이스 지우기로 이동합니다

## 8. 언어 옵션
   • 옵션 > 언어로 이동하여 19개의 지원 언어 간에 전환합니다:
     English, Türkçe, فارسی, 日本語, Español, Français, Deutsch, Italiano,
     Português (Brasil), Русский, 简体中文, 한국어, العربية, हिन्दी,
     Nederlands, Polski, Svenska, Norsk, Dansk
   • 인터페이스가 즉시 업데이트됩니다

## 9. 추가 옵션
   • "하위 폴더 포함": 선택한 폴더 내의 모든 하위 폴더를 스캔합니다
   • "방향이 다른 파일 건너뛰기": 세로 대 가로 일치를 무시합니다
   • "정확히 중복만 찾기": 정확히 중복된 파일만 찾으며, 유사한 파일은 표시되지 않습니다

## 10. 결과 저장 및 로드
    • 파일 > 결과 저장: 나중에 계속하기 위해 현재 결과를 저장합니다
    • 파일 > 결과 로드: 이전에 저장한 결과를 로드합니다

═══════════════════════════════════════════════════════════

## 팁:
• 마우스 휠 또는 +/- 버튼을 사용하여 미리보기 이미지를 확대/축소할 수 있습니다
• 파일을 마우스 오른쪽 버튼으로 클릭하여 열거나 위치를 확인합니다
• 키보드 탐색(화살표 키)을 사용하여 결과를 빠르게 탐색합니다
• 항목을 두 번 클릭하여 기본 이미지 뷰어에서 엽니다
