# COMPARATORE DI IMMAGINI - GUIDA UTENTE

Benvenuto in Image Comparator! Questa applicazione ti aiuta a trovare e gestire immagini duplicate nelle tue cartelle.

═══════════════════════════════════════════════════════════

## 1. AGGIUNGERE CARTELLE
   • Fai clic sul pulsante "Aggiungi Cartella" o trascina e rilascia le cartelle nell'elenco di output in basso
   • L'applicazione cercherà i file immagine nella cartella selezionata
   • Puoi aggiungere più cartelle per confrontare immagini in diverse posizioni

## 2. TROVARE DUPLICATI
   • Fai clic su "Trova Duplicati" per avviare il processo di confronto
   • L'applicazione analizzerà le immagini e troverà file simili o duplicati
   • Il progresso verrà mostrato nella barra di avanzamento
   • Puoi mettere in pausa o interrompere il processo in qualsiasi momento utilizzando i rispettivi pulsanti

## 3. COMPRENDERE I RISULTATI
   • Le coppie duplicate vengono mostrate in due elenchi affiancati
   • Gli indicatori codificati per colore mostrano i livelli di confidenza:
     - Blu: Duplicato
     - Rosso: Alta confidenza di somiglianza
     - Giallo: Media confidenza di somiglianza
     - Verde: Bassa confidenza di somiglianza
   • Fai clic su qualsiasi file per visualizzarlo in anteprima nel pannello centrale

## 4. FORMATI DI RICERCA
   • Vai su Opzioni > Formati di Ricerca per selezionare quali tipi di immagine scansionare
   • Formati supportati: JPEG, BMP, PNG, GIF, TIFF, ICO
   • Tutti i formati sono abilitati per impostazione predefinita

## 5. OPZIONI DI ELIMINAZIONE
   • Seleziona i file da eliminare utilizzando le caselle di controllo
   • "Elimina Selezionati": Rimuove i file selezionati dal tuo sistema
   • "Segna per Eliminazione": Evidenzia i file per l'eliminazione successiva (mostrati in rosso)
   • "Rimuovi dall'Elenco": Rimuove gli elementi dai risultati senza eliminare i file

## 6. METODO DI ELIMINAZIONE
   • Vai su Opzioni > Metodo di Eliminazione per scegliere:
     - "Invia al Cestino": Opzione più sicura, consente il recupero dei file (predefinito)
     - "Elimina Permanentemente": I file vengono eliminati permanentemente senza opzione di recupero

## 7. FALSI POSITIVI
   • Se due immagini sono erroneamente abbinate, selezionale e fai clic su "Segna Come Falso Positivo"
   • Queste coppie verranno ricordate e non appariranno nelle scansioni future
   • Per cancellare il database dei falsi positivi, vai su Opzioni > Cancella Database Falsi Positivi

## 8. OPZIONI LINGUA
   • Vai su Opzioni > Lingua per passare tra le 18 lingue supportate:
     English, Türkçe, 日本語, Español, Français, Deutsch, Italiano,
     Português (Brasil), Русский, 简体中文, 한국어, العربية, हिन्दी,
     Nederlands, Polski, Svenska, Norsk, Dansk
   • L'interfaccia si aggiornerà immediatamente

## 9. OPZIONI AGGIUNTIVE
   • "Includi Sottocartelle": Scansiona tutte le sottocartelle all'interno delle cartelle selezionate
   • "Salta File con Orientamento Diverso": Ignora le corrispondenze verticale vs orizzontale
   • "Trova Solo Duplicati Esatti": Trova solo file duplicati esatti, i file simili non verranno mostrati

## 10. SALVARE E CARICARE RISULTATI
    • File > Salva Risultati: Salva i tuoi risultati attuali per continuare più tardi
    • File > Carica Risultati: Carica risultati salvati in precedenza

═══════════════════════════════════════════════════════════

## SUGGERIMENTI:
• Puoi ingrandire/ridurre le immagini di anteprima utilizzando la rotella del mouse o i pulsanti +/-
• Fai clic con il pulsante destro sui file per aprirli o visualizzare la loro posizione
• Usa la navigazione da tastiera (tasti freccia) per sfogliare rapidamente i risultati
• Fai doppio clic sugli elementi per aprirli nel tuo visualizzatore di immagini predefinito

Per ulteriori informazioni o per segnalare problemi, visita il repository del progetto.
