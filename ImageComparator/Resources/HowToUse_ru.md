# СРАВНЕНИЕ ИЗОБРАЖЕНИЙ - РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

Добро пожаловать в Image Comparator! Это приложение помогает находить и управлять дубликатами изображений в ваших папках.

═══════════════════════════════════════════════════════════

## 1. ДОБАВЛЕНИЕ ПАПОК
   • Нажмите кнопку "Добавить папку" или перетащите папки в список вывода внизу
   • Приложение просканирует выбранную папку на наличие файлов изображений
   • Вы можете добавить несколько папок для сравнения изображений в разных местах

## 2. ПОИСК ДУБЛИКАТОВ
   • Нажмите "Найти дубликаты", чтобы начать процесс сравнения
   • Приложение проанализирует изображения и найдет похожие или дублирующиеся файлы
   • Прогресс будет показан на индикаторе выполнения
   • Вы можете приостановить или остановить процесс в любое время, используя соответствующие кнопки

## 3. ПОНИМАНИЕ РЕЗУЛЬТАТОВ
   • Дублирующиеся пары показаны в двух списках рядом
   • Цветные индикаторы показывают уровни достоверности:
     - Синий: Дубликат
     - Красный: Высокая степень сходства
     - Желтый: Средняя степень сходства
     - Зеленый: Низкая степень сходства
   • Щелкните по любому файлу, чтобы просмотреть его на центральной панели

## 4. ФОРМАТЫ ПОИСКА
   • Перейдите в Параметры > Форматы поиска, чтобы выбрать, какие типы изображений сканировать
   • Поддерживаемые форматы: JPEG, BMP, PNG, GIF, TIFF, ICO
   • Все форматы включены по умолчанию

## 5. ПАРАМЕТРЫ УДАЛЕНИЯ
   • Выберите файлы для удаления с помощью флажков
   • "Удалить выбранные": Удаляет отмеченные файлы из вашей системы
   • "Пометить для удаления": Выделяет файлы для последующего удаления (показаны красным)
   • "Удалить из списка": Удаляет элементы из результатов без удаления файлов

## 6. СПОСОБ УДАЛЕНИЯ
   • Перейдите в Параметры > Способ удаления, чтобы выбрать:
     - "Отправить в корзину": Более безопасный вариант, позволяет восстановить файлы (по умолчанию)
     - "Удалить навсегда": Файлы удаляются безвозвратно без возможности восстановления

## 7. ЛОЖНЫЕ СРАБАТЫВАНИЯ
   • Если два изображения неправильно сопоставлены, выберите их и нажмите "Пометить как ложное срабатывание"
   • Эти пары будут запомнены и не появятся в будущих сканированиях
   • Чтобы очистить базу данных ложных срабатываний, перейдите в Параметры > Очистить базу данных ложных срабатываний

## 8. ЯЗЫКОВЫЕ ПАРАМЕТРЫ
   • Перейдите в Параметры > Язык для переключения между 19 поддерживаемыми языками:
     English, Türkçe, فارسی, 日本語, Español, Français, Deutsch, Italiano,
     Português (Brasil), Русский, 简体中文, 한국어, العربية, हिन्दी,
     Nederlands, Polski, Svenska, Norsk, Dansk
   • Интерфейс обновится немедленно

## 9. ДОПОЛНИТЕЛЬНЫЕ ПАРАМЕТРЫ
   • "Включить вложенные папки": Сканирует все подпапки в выбранных папках
   • "Пропускать файлы с разной ориентацией": Игнорирует совпадения портретной и альбомной ориентации
   • "Искать только точные дубликаты": Находит только точные дублирующиеся файлы, похожие файлы показаны не будут

## 10. СОХРАНЕНИЕ И ЗАГРУЗКА РЕЗУЛЬТАТОВ
    • Файл > Сохранить результаты: Сохраняет ваши текущие находки для продолжения позже
    • Файл > Загрузить результаты: Загружает ранее сохраненные результаты

## СОВЕТЫ:
• Вы можете увеличивать/уменьшать изображения предварительного просмотра с помощью колеса мыши или кнопок +/-
• Щелкните правой кнопкой мыши по файлам, чтобы открыть их или просмотреть их местоположение
• Используйте навигацию с клавиатуры (клавиши со стрелками) для быстрого просмотра результатов
• Дважды щелкните по элементам, чтобы открыть их в вашей программе просмотра изображений по умолчанию

Для получения дополнительной информации или сообщения о проблемах посетите репозиторий проекта.
