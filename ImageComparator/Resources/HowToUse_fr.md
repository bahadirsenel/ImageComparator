# COMPARATEUR D'IMAGES - GUIDE DE L'UTILISATEUR

Bienvenue dans Image Comparator ! Cette application vous aide à trouver et à gérer les images en double dans vos dossiers.

═══════════════════════════════════════════════════════════

## 1. AJOUTER DES DOSSIERS
   • Cliquez sur le bouton "Ajouter un Dossier" ou glissez-déposez des dossiers dans la liste de sortie en bas
   • L'application recherchera les fichiers image dans le dossier sélectionné
   • Vous pouvez ajouter plusieurs dossiers pour comparer des images à différents emplacements

## 2. RECHERCHER DES DOUBLONS
   • Cliquez sur "Rechercher des Doublons" pour démarrer le processus de comparaison
   • L'application analysera les images et trouvera des fichiers similaires ou en double
   • La progression sera affichée dans la barre de progression
   • Vous pouvez mettre en pause ou arrêter le processus à tout moment en utilisant les boutons respectifs

## 3. COMPRENDRE LES RÉSULTATS
   • Les paires en double sont affichées dans deux listes côte à côte
   • Les indicateurs codés par couleur montrent les niveaux de confiance :
     - Bleu : Doublon
     - Rouge : Haute confiance de similitude
     - Jaune : Confiance moyenne de similitude
     - Vert : Faible confiance de similitude
   • Cliquez sur n'importe quel fichier pour le prévisualiser dans le panneau central

## 4. FORMATS DE RECHERCHE
   • Allez dans Options > Formats de Recherche pour sélectionner les types d'image à analyser
   • Formats pris en charge : JPEG, BMP, PNG, GIF, TIFF, ICO
   • Tous les formats sont activés par défaut

## 5. OPTIONS DE SUPPRESSION
   • Sélectionnez les fichiers à supprimer en utilisant les cases à cocher
   • "Supprimer la Sélection" : Supprime les fichiers cochés de votre système
   • "Marquer pour Suppression" : Met en évidence les fichiers pour une suppression ultérieure (affichés en rouge)
   • "Retirer de la Liste" : Retire les éléments des résultats sans supprimer les fichiers

## 6. MÉTHODE DE SUPPRESSION
   • Allez dans Options > Méthode de Suppression pour choisir :
     - "Envoyer à la Corbeille" : Option plus sûre, permet la récupération des fichiers (par défaut)
     - "Supprimer Définitivement" : Les fichiers sont supprimés définitivement sans option de récupération

## 7. FAUX POSITIFS
   • Si deux images sont incorrectement appariées, sélectionnez-les et cliquez sur "Marquer comme Faux Positif"
   • Ces paires seront mémorisées et n'apparaîtront pas dans les analyses futures
   • Pour effacer la base de données des faux positifs, allez dans Options > Effacer la Base de Données des Faux Positifs

## 8. OPTIONS DE LANGUE
   • Allez dans Options > Langue pour basculer entre 19 langues prises en charge :
     English, Türkçe, فارسی, 日本語, Español, Français, Deutsch, Italiano,
     Português (Brasil), Русский, 简体中文, 한국어, العربية, हिन्दी,
     Nederlands, Polski, Svenska, Norsk, Dansk
   • L'interface sera mise à jour immédiatement

## 9. OPTIONS SUPPLÉMENTAIRES
   • "Inclure les Sous-dossiers" : Analyse tous les sous-dossiers dans les dossiers sélectionnés
   • "Ignorer les Fichiers avec Orientation Différente" : Ignore les correspondances portrait vs paysage
   • "Trouver Uniquement les Doublons Exactes" : Ne trouve que les fichiers en double exacts, les fichiers similaires ne seront pas affichés

## 10. ENREGISTRER ET CHARGER LES RÉSULTATS
    • Fichier > Enregistrer les Résultats : Sauvegarde vos résultats actuels pour continuer plus tard
    • Fichier > Charger les Résultats : Charge les résultats précédemment enregistrés

═══════════════════════════════════════════════════════════

## CONSEILS :
• Vous pouvez zoomer sur les images d'aperçu en utilisant la molette de la souris ou les boutons +/-
• Cliquez avec le bouton droit sur les fichiers pour les ouvrir ou afficher leur emplacement
• Utilisez la navigation au clavier (touches fléchées) pour parcourir rapidement les résultats
• Double-cliquez sur les éléments pour les ouvrir dans votre visionneuse d'images par défaut

Pour plus d'informations ou pour signaler des problèmes, visitez le dépôt du projet.
