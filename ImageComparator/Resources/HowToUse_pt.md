# COMPARADOR DE IMAGENS - GUIA DO USUÁRIO

Bem-vindo ao Image Comparator! Este aplicativo ajuda você a encontrar e gerenciar imagens duplicadas em suas pastas.

═══════════════════════════════════════════════════════════

## 1. ADICIONAR PASTAS
   • Clique no botão "Adicionar Pasta" ou arraste e solte pastas na lista de saída na parte inferior
   • O aplicativo irá procurar arquivos de imagem na pasta selecionada
   • Você pode adicionar várias pastas para comparar imagens em diferentes locais

## 2. ENCONTRAR DUPLICATAS
   • Clique em "Encontrar Duplicatas" para iniciar o processo de comparação
   • O aplicativo irá analisar imagens e encontrar arquivos similares ou duplicados
   • O progresso será mostrado na barra de progresso
   • Você pode pausar ou parar o processo a qualquer momento usando os botões respectivos

## 3. ENTENDER OS RESULTADOS
   • Pares duplicados são mostrados em duas listas lado a lado
   • Indicadores codificados por cor mostram níveis de confiança:
     - Azul: Duplicado
     - Vermelho: Alta confiança de similaridade
     - Amarelo: Média confiança de similaridade
     - Verde: Baixa confiança de similaridade
   • Clique em qualquer arquivo para visualizá-lo no painel central

## 4. FORMATOS DE PESQUISA
   • Vá para Opções > Formatos de Pesquisa para selecionar quais tipos de imagem escanear
   • Formatos suportados: JPEG, BMP, PNG, GIF, TIFF, ICO
   • Todos os formatos estão habilitados por padrão

## 5. OPÇÕES DE EXCLUSÃO
   • Selecione os arquivos para excluir usando as caixas de seleção
   • "Excluir Selecionados": Remove arquivos marcados do seu sistema
   • "Marcar Para Exclusão": Destaca arquivos para exclusão posterior (mostrados em vermelho)
   • "Remover da Lista": Remove itens dos resultados sem excluir arquivos

## 6. MÉTODO DE EXCLUSÃO
   • Vá para Opções > Método de Exclusão para escolher:
     - "Enviar Para Lixeira": Opção mais segura, permite recuperação de arquivos (padrão)
     - "Excluir Permanentemente": Arquivos são excluídos permanentemente sem opção de recuperação

## 7. FALSOS POSITIVOS
   • Se duas imagens forem correspondidas incorretamente, selecione-as e clique em "Marcar Como Falso Positivo"
   • Esses pares serão lembrados e não aparecerão em verificações futuras
   • Para limpar o banco de dados de falsos positivos, vá para Opções > Limpar Banco de Dados de Falsos Positivos

## 8. OPÇÕES DE IDIOMA
   • Vá para Opções > Idioma para alternar entre 19 idiomas suportados:
     English, Türkçe, 日本語, Español, Français, Deutsch, Italiano,
     Português (Brasil), Русский, 简体中文, 한국어, العربية, हिन्दी,
     Nederlands, Polski, Svenska, Norsk, Dansk
   • A interface será atualizada imediatamente

## 9. OPÇÕES ADICIONAIS
   • "Incluir Subpastas": Escaneia todas as subpastas dentro das pastas selecionadas
   • "Pular Arquivos Com Orientação Diferente": Ignora correspondências de retrato vs paisagem
   • "Encontrar Apenas Duplicatas Exatas": Encontra apenas arquivos duplicados exatos, arquivos similares não serão mostrados

## 10. SALVAR E CARREGAR RESULTADOS
    • Arquivo > Salvar Resultados: Salva suas descobertas atuais para continuar mais tarde
    • Arquivo > Carregar Resultados: Carrega resultados salvos anteriormente

═══════════════════════════════════════════════════════════

## DICAS:
• Você pode aumentar/diminuir o zoom nas imagens de visualização usando a roda do mouse ou os botões +/-
• Clique com o botão direito nos arquivos para abri-los ou visualizar sua localização
• Use a navegação por teclado (teclas de seta) para navegar pelos resultados rapidamente
• Clique duas vezes nos itens para abri-los em seu visualizador de imagens padrão

Para mais informações ou para reportar problemas, visite o repositório do projeto.
